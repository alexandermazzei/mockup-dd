API Documentation

Build custom integrations with our comprehensive API. Access data, trigger workflows, and extend functionality.
DueDiligenceAI Integration API v2.0
Use our RESTful API to create custom integrations and extend platform capabilities.
GET /api/v2/integrations
Retrieve all available and connected integrations for your account.
// Example Request GET /api/v2/integrations HTTP/1.1 Host: api.duediligenceai.com Authorization: Bearer YOUR_API_KEY
// Example Response { "status": "success", "data": { "integrations": [ { "id": "int_12345", "name": "Salesforce", "type": "crm", "status": "connected", "last_sync": "2025-05-07T12:30:45Z" }, { "id": "int_67890", "name": "SAP ERP", "type": "erp", "status": "connected", "last_sync": "2025-05-07T10:15:22Z" } ] } }
POST /api/v2/integrations/connect
Initialize a new integration connection with specified platform.
// Example Request POST /api/v2/integrations/connect HTTP/1.1 Host: api.duediligenceai.com Authorization: Bearer YOUR_API_KEY Content-Type: application/json { "integration_id": "hubspot", "credentials": { "api_key": "YOUR_HUBSPOT_API_KEY" }, "settings": { "sync_frequency": "hourly", "data_types": ["contacts", "companies", "deals"] } }

Parameters

Parameter Type Description
integration_id Required string Identifier for the integration platform
credentials Required object Authentication credentials for the integration
settings object Configuration settings for the integration
POST /api/v2/data/import
Import data from a connected integration into your DueDiligenceAI project.
// Example Request POST /api/v2/data/import HTTP/1.1 Host: api.duediligenceai.com Authorization: Bearer YOUR_API_KEY Content-Type: application/json { "integration_id": "int_12345", "project_id": "proj_67890", "data_types": ["accounts", "contacts", "opportunities"], "filters": { "created_after": "2024-01-01T00:00:00Z", "status": ["active", "qualified"] } }
PUT /api/v2/integrations/:integration_id
Update configuration or credentials for an existing integration.
DELETE /api/v2/integrations/:integration_id
Disconnect and remove an integration from your account.
